Region-wise Causal Intervention on Epistemically Uncertain Visual Tokens

A region-wise causal intervention framework for understanding which semantic image regions influence object hallucination in LLaVA-1.5 and Qwen2.5-VL.

Research Question

Where does uncertain visual evidence matter?

When an object is removed, hallucination may be driven by residual evidence near the original object location, nearby contextual structure, or broader scene-level cues.

01

Removed region

Does uncertainty at the original object location directly influence the hallucinated answer?

02

Context region

Do nearby objects and local scene structure reinforce the absent object's presence?

03

Background region

Do global scene priors and co-occurrence cues bias the model toward confirming the removed object?

Method

From removed objects to causal interventions

The pipeline combines controlled counterfactual images, semantic region maps, representation-space perturbations, epistemic uncertainty estimation, and region-specific visual-token suppression.

1

Counterfactual construction

One target object is removed from each COCO image using LaMa inpainting.

2

Semantic token mapping

Image regions are aligned with fixed LLaVA patches or dynamic Qwen merged visual tokens.

3

Epistemic uncertainty

Clean and adversarially perturbed visual representations are compared to identify unstable visual tokens.

4

Causal masking

None, global, removed-region, context-region, and background-region interventions are evaluated.

LLaVA-1.5-7B

Fixed visual grid

24 × 24 visual tokens with suppression inside selected visual-attention layers.

LLaVA-1.5-13B

Within-family comparison

The same intervention is evaluated at a larger model scale.

Qwen2.5-VL-7B

Dynamic merged tokens

Variable visual grids with merged-token suppression before language-model input.

Control Experiments

Are the improvements specific to epistemic uncertainty?

Matched random and matched low-uncertainty controls distinguish uncertainty-guided token selection from general visual-information removal.

Random

General information removal matters

Randomly suppressing the same number of tokens often produces effects comparable to high-uncertainty masking.

Low uncertainty

Architecture-dependent specificity

LLaVA-1.5-7B differs from matched low-uncertainty masking, whereas Qwen2.5-VL-7B does not.

Active masks

Weak regional effects remain weak

Removed-region and context-region effects remain limited even when analysis is restricted to samples with active suppression.

Recognition Trade-off

Hallucination reduction is not cost-free

Global and background masking also reduce correct object recognition when the queried object is present in the original image.

Central Takeaway

Region-wise masking reveals where visual evidence influences hallucination, but uncertainty ranking is not universally selective.

Key Findings

What the experiments establish

01

Global and background masking are strongest

These interventions produce the largest hallucination reductions in two distinct 7B model architectures.

02

The effect is model-dependent

LLaVA-1.5-13B does not reproduce the positive masking effects observed in the two 7B models.

03

Uncertainty is not uniquely causal

Matched controls show that uncertainty ranking is not universally responsible for the observed improvements.

04

Mitigation has a recognition cost

The same masking conditions that reduce hallucination can also remove useful visual evidence.
